# PR Test

这是一个用来测试 Pull Request 的仓库。

## 如何参与

1. Fork 这个仓库到你的账号
2. Clone 你的 fork 到本地
3. 创建新分支
4. 做一些修改
5. 推送到你的 fork
6. 创建 Pull Request 到这个仓库

## 快速开始

```bash
# 克隆你的 fork
git clone https://github.com/YOUR_USERNAME/pr-test.git

# 创建新分支
git checkout -b my-feature

# 做一些修改
echo "Hello from my feature" >> hello.txt

# 提交并推送
git add .
git commit -m "Add my feature"
git push origin my-feature
```

然后去 GitHub 创建 PR！
